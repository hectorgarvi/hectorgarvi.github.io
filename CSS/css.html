Aqui presentaré todos los ejercicios realizados hasta ahora de CSS.

Ejercicio 1. Banderas.

En este ejercicio, elaboraremos mediante divs diversas banderas de varios paises, ajustando el color con gradientes.




Ejercicio 2. Bordes de párrafos.

En este ejercicio, elaboramos diferentes párrafos y les damos bordes de diferentes estilos, para así ver la gran variedad de los mismos.




Ejercicio 3. Mundo Suzuki.

En este ejercicio hemos elaborado una página web siguiendo el modelo que nos daba las diapositivas, haciendo la página responsive y con background personalizado.




Ejercicio 4. Disposición de divs con Float.

En este ejercicio nos pedía distribuir ciertas cajas a lo largo de la página, utilizando una plantilla y diferentes divs.




Ejercicio 5. Horario.

En este ejercicio hemos realizado nuestro horario en código HTML y le hemos ańadido estilos a las diferentes asignaturas, así como al fondo, para personalizarla a nuestro gusto.




Ejercicio 6. Divs anidados.

En este ejercicio hemos dado propiedades diversas a diferentes divs para que se superpongan, ańadiendo bordes, márgenes y colores al fondo.




Ejercicio 7. Manejando un div.

Aquí hemos dado propiedades a un div para ańadir un borde y un outline a la imagen junto con un padding.




Ejercicio 8. Box model con 6 imagenes.

Hemos realizado un box model de 6 imágenes dando diferentes atributos.




Ejercicio 9. Posicionamiento.

En este ejercicio nos pedía elaborar una página en la cual agregásemos varias imágenes con posicionamientos diferentes.




Ejercicio 10. Fuente y texto.

En este ejercicio, habremos introducido propiedades para cambiar la fuente y color del texto.




Ejercicio 11. Color en el body.

En este ejercicio habremos introducido propiedades para cambiar el color del body.
